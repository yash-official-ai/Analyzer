# 📊 NumPy Analyzer

A **menu-driven Python project** built using the **NumPy** library that allows users to create arrays, perform mathematical operations, combine/split arrays, search and sort data, and compute statistical measures. This project is designed to strengthen the understanding of NumPy fundamentals through an interactive command-line interface.

---

## 🚀 Features

- ✅ Create NumPy Arrays
  - 1D Array
  - 2D Array
  - 3D Array

- ✅ Perform Mathematical Operations
  - Addition
  - Subtraction
  - Multiplication
  - Division

- ✅ Combine or Split Arrays
  - Vertical Stack (Combine Arrays)
  - Split Arrays

- ✅ Search, Sort & Filter Arrays
  - Search a Value
  - Sort Array
  - Filter Values

- ✅ Compute Aggregates & Statistics
  - Sum
  - Mean
  - Median
  - Standard Deviation
  - Variance
